# ТЗ: Агент 2 — Автоматический онбординг + Агент 3 — Контроль обучения

## Агент 2: Онбординг

### Концепция
При добавлении нового сотрудника в команду — агент автоматически назначает план обучения на основе должности и отправляет приветствие в Telegram.

### Шаг 1: API триггер при добавлении сотрудника
В существующий API создания сотрудника (найти через grep "team" или "employees" в app/api) добавить after-insert хук:
1. Найти learning plans для tenant где название или теги совпадают с должностью/отделом сотрудника
2. Если найдены — автоматически назначить (INSERT в learning_plan_assignments или аналогичную таблицу)
3. Создать notification: "Новому сотруднику [имя] назначен план обучения [название]"
4. Если у компании подключён Telegram бот и у сотрудника есть telegramChatId — отправить приветствие

### Шаг 2: API для ручного назначения онбординга
Файл: app/api/modules/knowledge/onboarding/route.ts
POST: { userId, planId? } — назначить план (или автоподбор по должности)
GET: список сотрудников с их статусом онбординга

### Шаг 3: Telegram напоминания об обучении
Файл: app/api/cron/knowledge-reminders/route.ts
GET ?secret=CRON_SECRET:
1. Найти все незавершённые назначения где дедлайн < 3 дня
2. Для каждого сотрудника с telegramChatId — отправить напоминание
3. Текст: "У вас назначен курс [название]. Осталось [N] уроков. Дедлайн: [дата]"

## Агент 3: Контроль обучения

### Концепция
Еженедельный cron анализирует прогресс обучения и уведомляет руководителей.

### Шаг 1: Cron контроля
Файл: app/api/cron/knowledge-progress/route.ts
GET ?secret=CRON_SECRET:
1. Для каждого tenant собрать все активные назначения
2. Найти отстающих: дедлайн < 7 дней и прогресс < 50%
3. Найти просроченных: дедлайн прошёл и не завершено
4. Сгруппировать по руководителю
5. Отправить сводку каждому руководителю (notification + Telegram):
   "Еженедельный отчёт обучения: 5 завершили, 3 отстают, 1 просрочил"
6. Ответ: { ok, tenants, completed, behind, overdue }

### Шаг 2: Пятая строка accordion в settings
В knowledge-v2/settings/page.tsx добавить:
- Иконка: GraduationCap (lucide)
- Текст: "Контроль обучения"
- Badge: "X отстают" оранжевый / "Все в графике" зелёный
- При раскрытии: таблица (сотрудник, план, прогресс %, дедлайн, статус)
- Кнопка "Отправить напоминания"
