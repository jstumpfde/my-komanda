# HireFlow — HR Рекрутинговая платформа

## Стек

- **Framework:** Next.js 16 (App Router) + React 19 + TypeScript
- **Styling:** Tailwind CSS 4 + CSS-переменные (oklch) + tw-animate-css
- **UI Kit:** shadcn/ui (стиль new-york) — ~50 компонентов в `components/ui/`
- **Icons:** Lucide React
- **Charts:** Recharts 2.15
- **Font:** Inter (latin + cyrillic)
- **Package manager:** pnpm
- **Language:** русский (весь UI на русском)

## Структура папок

```
app/
  layout.tsx          — корневой layout, ThemeProvider, шрифт Inter
  page.tsx            — главная страница (дашборд вакансии)

  dashboard/          — бизнес-компоненты дашборда
```

## Дизайн-система

### Темы

3 темы через CSS-классы на `<html>`, управляемые `next-themes`:

| Тема | Класс | Характер |
|------|-------|----------|
| Light | `:root` | Чистый белый, фиолетовый primary (oklch 0.54 0.16 280) |
| Dark | `.dark` | Тёмно-синий/серый фон, яркий фиолетовый primary |
| Warm | `.warm` | Тёплый бежевый фон, синий primary |

### Цветовые токены (CSS-переменные, oklch)

- `--primary` — фиолетовый/синий, основной акцент
- `--destructive` — красный, ошибки и отказы
- `--success` — зелёный, принятие и позитивные действия
- `--warning` — жёлтый/оранжевый, предупреждения и резерв
- `--muted` / `--muted-foreground` — приглушённые элементы
- `--sidebar` — отдельная палитра для боковой панели (в light — яркий синий фон)
- `--chart-1..5` — палитра для графиков

### Градиенты колонок воронки

Определены в `lib/column-config.ts`:
- **Новые:** `#22d3ee → #06b6d4` (бирюзовый)
- **Квалификация:** `#f59e0b → #d97706` (янтарный)
- **Тестирование:** `#3b82f6 → #06b6d4` (синий)
- **Интервью:** `#8b5cf6 → #7c3aed` (фиолетовый)
- **Предложение:** `#22c55e → #16a34a` (зелёный)

### Типографика

- Шрифт: Inter (cyrillic + latin)
- Размеры: `text-2xl` (заголовки), `text-sm` (основной текст), `text-xs`/`text-[11px]`/`text-[10px]` (мелкие элементы)
- Все тексты на русском языке

### Компоненты — соглашения

- shadcn/ui стиль `new-york` с кастомными CSS-переменными
- Скругления через `--radius: 0.5rem`
- Карточки: `rounded-lg border border-border bg-card`
- Кнопки действий: ghost-вариант с семантическими цветами (success/destructive/warning)
- Бейджи: `variant="outline"` с цветовыми классами по источнику/скору
- Градиентные полоски на заголовках колонок и бейджах статусов

## Данные и состояние

- **Нет бэкенда** — все данные захардкожены в `page.tsx` (9 кандидатов)
- **In-memory storage** — `vacancy-storage.ts` хранит категории вакансий (3 категории, ~10 вакансий)
- Состояние управляется через `useState` в `page.tsx` и пробрасывается пропсами
- Фильтрация кандидатов выполняется на клиенте в `page.tsx`

## Соглашения по коду

- Все компоненты — `"use client"` (нет серверных компонентов кроме layout)
- Именование файлов: kebab-case (`candidate-card.tsx`)
- Экспорт компонентов: named exports (`export function ComponentName`)
- Стилизация: Tailwind utility classes, `cn()` для условных классов
- Иконки: только Lucide React
- Alias-пути: `@/components`, `@/lib`, `@/hooks`
