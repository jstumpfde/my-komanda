# «Бизнес-ассистент» → Промокоды: разведка источников (04.07.2026)

Исследование для подпроекта «радар промокодов» (после Авиабилетов и Отелей).
Цель — понять, откуда реально брать действующие промокоды/скидки с комиссией,
и насколько реалистична приоритетная для Юрия ниша **бани/сауны и рестораны**
(кейс: промокод на баню до 45% вместо 15%).

## 1. Главный вывод (для решения)

Собственный «поисковик промокодов по всему рынку» строить не нужно — как и с
авиабилетами, реальный путь: **агрегировать готовые партнёрские фиды промокодов**
(легально, машиночитаемо, с комиссией) + курировать нишевые источники, где фида нет.

⚠️ **Важный нюанс (проверено):** единого «чистого» открытого API-фида промокодов
от самих брендов в РФ НЕТ ни в одной нише. Radario/Timepad ограничивают промокоды
своими каналами продаж; билетные/доставочные сервисы публикуют коды точечно.
Реальная инфраструктура — **партнёрские CPA-сети** (в первую очередь Admitad), где
промокоды — курируемый вторичный слой поверх реферальных ссылок. Значит «радар» =
Admitad-пул купонов + курируемый/скрейпинговый сбор точечных акций + Telegram-лента.

**Три реалистичных источника для MVP (по убыванию отдачи):**

1. **Admitad (партнёрская сеть) — фид промокодов.** Самый надёжный каркас.
   Admitad отдаёт вебмастеру **выгрузку купонов/промокодов** в CSV/XML/RSS +
   есть **Publisher API, метод «Купоны»** — описание акции, ссылка с меткой,
   кодовое слово, размер скидки, срок действия. Тысячи мерчантов РФ (маркетплейсы,
   доставка, услуги). Комиссия — как у любой CPA. Это машиночитаемый фид, который
   можно тянуть кроном и складывать в свою ленту (как flight_deals).
   - Publisher API «Купоны»: https://developers.admitad.com/ru/knowledge-base/article/%D0%BA%D1%83%D0%BF%D0%BE%D043D%D1%8B (см. также account.admitad.com/ru/developers/doc)
   - Помощь по купонам: https://help.admitad.com/ru/topic/10-kupony-i-promokody

2. **ePN / Cityads / Travelpayouts** — вторичные CPA-сети с похожими купон-инструментами
   (у Travelpayouts мы уже зарегистрируемся ради авиа/отелей — часть их промо-офферов
   можно переиспользовать для travel-промокодов). Дают охват тех мерчантов, кого нет
   в Admitad.

3. **Курируемые нишевые источники (бани/рестораны/развлечения)** — где партнёрского
   фида нет, лента наполняется полу-вручную или парсингом (как Telegram-лента у
   авиабилетов). Основной канал скидок в этой нише — **групповые купоны и банковский
   кэшбэк**, а не мерчант-промокоды (см. п.3).

## 2. Агрегаторы промокодов (РФ) с партнёркой/API

| Источник | API/фид | Профиль |
|---|---|---|
| **Admitad** | ✅ CSV/XML/RSS + Publisher API «Купоны» | Основной, тысячи мерчантов |
| ePN (admitad group) | ✅ купон-инструменты | Маркетплейсы, travel |
| Cityads | партнёрка, купоны | Игры, финансы, товары |
| Промокоды.ру / Купоника / Picodi | витрины (в осн. через Admitad-фиды) | Готовые витрины, не источник данных |

Практически все крупные RU-витрины промокодов сами тянут данные из Admitad/ePN —
т.е. подключившись к Admitad-фиду, мы получаем тот же массив, что и они, напрямую.

## 3. Бани/сауны и рестораны — реальность ниши (ключевой интерес Юрия)

Скидка «до 45% на баню» в РФ обычно приходит **НЕ через мерчант-промокод**, а через:

- **Групповые купоны (Biglion / Купонатор / Frendi)** — сайты дневных скидок:
  бани/сауны/СПА и рестораны со скидками **50–90%**. Это ровно механика «платить
  в разы меньше». Biglion исторически есть в CPA-сетях (партнёрка), т.е. можно
  тянуть его офферы и получать комиссию. Каталог бань/саун Biglion:
  https://www.biglion.ru/services/beauty/sauna , рестораны — раздел кафе/рестораны.
- **Банковский кэшбэк** — Т-Банк (Город: бронь столика + кэшбэк 5–15% в категории
  «Рестораны»), СберПрайм и т.п. Это не промокод, а возврат части суммы.
  https://www.tbank.ru/gorod/restaurants/
- **Агрегаторы акций ресторанов** — Restoclub (акции/скидки на сегодня, без API,
  но парсибельно): https://www.restoclub.ru/msk/news/actions ; бронь столиков —
  LeClick/SmartReserve (есть кэшбэк за брони): https://b2b.leclick.ru/

**Вывод по нише:** реалистично, но источник = **групповые купоны (Biglion) +
кэшбэк-сервисы + парсинг акций Restoclub**, а не «промокоды заведений». Для MVP
самое чистое — подключить Biglion через CPA-сеть (фид офферов бань/саун/ресторанов
со скидкой) и показывать их в разделе. Прямые «промокоды на баню» из Telegram-каналов
можно добавить курируемой лентой (как авиа-сливы).

## 4. Telegram-каналы промокодов/халявы (проверены WebFetch, на утверждение)

Механика та же, что у авиа-ленты (userbot GramJS + AI-разбор). Список утвердит
Юрий. Проверенные живые каналы (аудитория подтверждена через t.me/s/…):

Общие промокоды:
- **@freesind** «ХАЛЯВА | ПРОМОКОДЫ | СКИДКИ» — ~86K, чистые промокоды (Яндекс, Ozon, ВТБ)
- **@telewkin** «Скидосо» — ~76K, промокоды без розыгрышей
- **@sbermarket_kodi** «Купер/Сбермаркет» — ~62K, посты каждый час
- **@aktsiya_telegram** «Скидки, акции и промокоды» — ~23K (WB/Ozon/Маркет/Летуаль)
- **@lavka_skidok** — ~18K (маркетплейсы + партнёрские ссылки)

Еда/доставка:
- **@yandexedachannel** (официальный Яндекс Еда) — ~165K
- **@pizza_suchi_channel** «Скидки на еду» — ~5K (фастфуд/доставка)

Мероприятия/билеты:
- **@teatr_msk** «Театры и концерты со скидкой, Москва» — ~70K (партнёр stageon.ru)

⚠️ Ниша накручена — многие «билетные» каналы мёртвые/тестовые (@bilet_40 = 1 подписчик,
@tickets_moscow заброшен с 2022). Брать только проверенные, живость сверять перед
подключением. tgstat.ru блокирует автозапросы (403) — проверять через t.me/s/… и telemetr.me.

## 5. Рекомендация к реализации (когда дойдём до Промокодов)

- **Фаза 1:** подключить **Admitad Coupons feed/API** → крон тянет промокоды в
  таблицу `promo_deals` (аналог `flight_deals`), страница `/business-assistant/promo`
  с фильтром по категориям. Даёт сразу широкий охват + комиссию.
- **Фаза 2:** добавить **Biglion/групповые купоны** для ниши бани/рестораны/СПА
  (то, чем Юрий больше всего интересовался) — отдельная категория/вкладка.
- **Фаза 3:** курируемая Telegram-лента промо-сливов (userbot, как у авиа).

**Нужно от Юрия (позже, оргшаги):** регистрация вебмастером в **Admitad**
(admitad.com, подтверждение площадки) + при желании в Biglion-партнёрке. Отдельные
ключи, не пересекаются с Travelpayouts.
