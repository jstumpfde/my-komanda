## Задачи Phase 1

### T1. Починить пустой Sheet настроек блока
- Симптом: клик по шестерёнке блока → панель открывается, но тело пустое (только
  заголовок + неактивная «Сохранить»). Ширина 672px — НОРМА (не CSS-ширина).
- Гипотеза (PHASE0): схлопывание высоты в `funnel-builder.tsx:619` (SheetContent
  `overflow-y-auto flex flex-col`) + тело `:632` (`flex-1`). НО возможно Comp просто
  рендерит мало контента для некоторых блоков — НУЖНА БРАУЗЕР-ДИАГНОСТИКА.
- ⚠️ Браузер-проверка блокируется таб-гардом (Q1). Сначала решить Q1 ИЛИ открывать
  настройки на ЧЕРНОВИКЕ-вакансии (не активной — там гард не форсит «Кандидаты»).
- Действие: воспроизвести в браузере, измерить тело (height/children), глянуть консоль,
  затем точечный фикс. Проверить, что РАБОЧИЕ блоки не сломались.

  редактор теста). Различать с `test_task`.
